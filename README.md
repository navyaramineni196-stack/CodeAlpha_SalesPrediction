# Sales Prediction using Machine Learning

## CodeAlpha Internship Project

## Project Description

This project predicts sales using Machine Learning based on advertising expenditure. The model analyzes the impact of TV, Radio, and Newspaper advertising on sales and predicts future sales values.

## Dataset

Dataset used: Advertising Dataset

### Input Features:
- TV Advertising Spend
- Radio Advertising Spend
- Newspaper Advertising Spend

### Target:
- Sales

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Machine Learning Model

Algorithm Used:

**Linear Regression**

## Project Steps

1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Data Visualization
5. Feature Selection
6. Model Training
7. Sales Prediction
8. Model Evaluation

## Model Performance

R2 Score: 0.8994

The model achieved approximately 90% prediction accuracy.

## Results

- Advertising expenditure has a positive impact on sales.
- The model successfully predicts sales values.
- The analysis helps understand advertising effectiveness.

## Repository Files

- `Sales_Prediction.ipynb` - Complete project notebook
- `Advertising.csv` - Dataset
- `README.md` - Project documentation

## Conclusion

This project demonstrates the use of Machine Learning for sales forecasting and helps businesses make better decisions using data analysis.
